# Dynamic Data Table Manager

This project is a React + Redux + Material UI based web application that allows users to dynamically manage tabular data.  
It supports adding, editing, deleting, reordering rows and columns, global searching, pagination, and saving column visibility preferences.

## Overview

The goal of this project was to create a functional and responsive data table manager with clean UI and efficient state management.  
It is built using React, Redux Toolkit, and Material UI, with TypeScript for better maintainability.

## Features

### Core Features
- Table with default columns: Name, Email, Age, Role
- Column sorting (ascending and descending)
- Global search across all columns
- Client-side pagination (10 rows per page)
- Dynamic columns (add new, hide or show)
- Column visibility persistence using localStorage or Redux Persist
- Import and export CSV functionality

### Additional Features
- Inline editing for rows with save and cancel options
- Row deletion with confirmation dialog
- Drag-and-drop for reordering rows and columns
- Responsive design that works on all screen sizes
- Light and dark theme support using Material UI
